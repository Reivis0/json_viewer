# Графический просмотрщик JSON

## Описание задачи 
Разработать приложение для просмотра текстовых файлов в формате JSON в виде интерактивного дерева.

## Интерфейс
- _Область для текстового ввода данных в формате JSON_. Для данной области доступно непосредственное редактирование, а также копирование и вставка из буфера обмена. Кроме того, облать должна обеспечивать простейшую подсветку синтаксиса JSON формата.
- _Область древовидного представления JSON данных._ Область нередактируема, однако позволяет сворачивать и разворачивать отдельные иерархические уровни.
- _Кнопку открытия данных._ При ее активации пользовател. предоставляется возможность выбрать файл для загрузки в облать текстового ввода.
- _Кнопку сворачивания/разворачивания всех уровней иерархии._ Действие кнопки зависит от текущего состояния древовидного представления. При наличии свернутых узлов, кнопка разворачивает все узлы дерева. Если все узлы развернуты, то кнопка их сворачивает.
- _Кнопку обновления древовидного представления._ При ее активации представление актуализируется в соответствии с данными из области текстового ввода.

## Требования
- Приложение должно использовать библиотеку Qt5 (предлагается 5.15.x)
- Сборка приложения должна осуществляться с использованием системы cmake
- Интерфейс приложения, подсветка синтаксиса, парсинг JSON формата должны быть реализованы с помощью средств используемой библиотеки Qt
- Для реализации древовидного представления JSON необходимо использовать отдельно класс модели (наследника QAbstractItemModel) и класс QTreeView (или его наследника)
- Проект должен содержать автоматические тесты (как минимум для класса модели древовидного представления и функций парсинга JSON), написанные с использованием Google Test
- Сообщения о нештатных ситуациях (ошибки парсинга, невозможность открыть файл и т.п.) должны выводится в консоль с помощью функций семейства qDebug()
